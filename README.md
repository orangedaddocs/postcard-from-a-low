# Orange Dad Docs

Interactive Bitcoin pages and planning notes by Orange Dad, from the maker of [genxbitcoin.io](https://genxbitcoin.io). Local-first, no tracking, receipts always.

## Pages

- **[power-law-catch-up](power-law-catch-up/)** — Two AI models, one trendline: how long does the discount last? Four catch-up scenarios from the June 2026 drawdown back to the Power Law 1.0x line.
- **[dca-110](dca-110/)** — DCA 110 calculator. Stack an extra $110 a month, week, or day starting July 1, 2026, until each scenario touches the trendline, and see what it buys.

Every page is a single self-contained HTML file with no build step and no dependencies beyond Google Fonts. Open it in a browser and it works.

Not financial advice. Just a dad doing the math out loud.
