# DCA 110 · A Postcard from a Bitcoin Low

One standalone page by Orange Dad. Local-first, no tracking, receipts always.

**Live page:** https://orangedaddocs.github.io/dca-110/

Two parts on one link:

1. **The chart note** — Two AI models, one trendline: how long does the discount last? Four catch-up scenarios from the June 2026 drawdown ($61.5K spot, 0.46x of the Power Law line) back to trend.
2. **The calculator** — DCA 110. Stack an extra $110 a month, week, or day starting July 1, 2026, until each scenario touches the trendline, and see what it buys.

Why 110? Summer 2026 is the summer of the BIP-110 debate. Orange Dad is for the fork, but this page isn't about it — the number is the timestamp.

The whole thing is a single self-contained HTML file ([index.html](index.html)) with no build step and no dependencies beyond Google Fonts. Open it in a browser and it works. The two parts originally lived as separate pages; they're preserved in the first commit of this repo.

Not financial advice. Just a dad doing the math out loud.
